# VSBSplitter
IntrepidCS VSB File Splitter

*本程序基于QT编写*


#### 程序功能:

将一个大的VSB文件分割成多个小VSB文件 
> 注意： 当前本工具仅支持0x102的VSB格式: [Vehicle Spy Binary File Buffer File Spec](https://cdn.intrepidcs.net/support/VehicleSpy/vehiclespyhelpdoc.html)
* 基于给定的文件大小分割
* 基于给定的报文条数分割（即每个小文件中包含指定的报文条数）
